Colophone is your ebook library made awesome: browse your tags nice and clean, show series stats, click and read your favourite books all in one place.

## Installation

Install from the [releases page](https://github.com/gennaro-tedesco/metabrowser/releases) choosing your architecture.

## Development

Build locally with

```sh
wails build && open /build/bin/colophon.app
```
